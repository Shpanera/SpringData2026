# КОМПЛЕКСНОЕ РАЗВИТИЕ ТЕРРИТОРИЙ, ПРИЛЕГАЮЩИХ К ТПУ

Цифровой сервис для комплексной оценки и планирования развития территорий, прилегающих к транспортно-пересадочным узлам (ТПУ). Проект разрабатывается в рамках выпускной квалификационной работы (магистратура/специалитет).

**Ключевая идея**: автоматизация выявления пространственных дисбалансов между транспортной доступностью, функциональным насыщением и морфологией застройки для формирования обоснованных градостроительных рекомендаций на принципах транзитно-ориентированного развития (TOD).

## Цель ВКР
Цель ВКР: разработка специализированного цифрового сервиса для комплексной оценки и планирования развития территорий, прилегающих к транспортно-пересадочным узлам (ТПУ), с целью повышения качества и скорости процесса оценки территорий.
## Методологическая основа
| Метод                              | Описание                                          | Источник             |
|------------------------------------|---------------------------------------------------|----------------------|
| TOD (Transit-Oriented Development) | Транзитно-ориентированное проектирование          | ITDP Standard        |
| Spacematrix                        | Морфологический анализ плотности и типа застройки | Meta Berghauser Pont |
| Mixed-use Index (MXI)              | Оценка функционального разнообразия               | Van den Hoek         |
| Матрица диспропорций               | Выявление дисбалансов "насыщенность ↔ связность"  | Гутнов А.Э.          |
